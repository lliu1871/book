# 第209章  剥崔颢黄鹤楼诗吊大学生 

　　此诗见于《伪自由书·崇实》。 

　　阔人已骑文化去，此地空余文化城。 

　　文化一去不复返，古城千载冷清清。 

　　专车队队前门站，晦气重重大学生。 

　　日薄榆关何处抗，烟花场上没人惊。
